Welcome to the **Examples and Demos** section of the HAPI MCP documentation! Here, you'll find practical guides and walkthroughs on how to set up and utilize HAPI MCP servers using various OpenAPI specifications (OAS). These examples demonstrate how to integrate different APIs into your AI agents and applications seamlessly.

Each example provides step-by-step instructions, from installing the HAPI MCP CLI to deploying your MCP server. Whether you're looking to create a server for OpenAI, LinkedIn, or any other API, these guides will help you get started quickly and efficiently.

Enjoy exploring the possibilities with HAPI MCP!
